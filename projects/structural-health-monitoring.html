Structural Health Monitoring — Piezoelectric Sensing System

A proof-of-concept non-destructive inspection system that uses piezoelectric transducers, ultrasonic pulses, and signal analysis to identify changes caused by structural damage.

Project photos will be added when available

System overview

  • Instrumented structural specimens with piezoelectric crystals.
  • Transmitted ultrasonic pulses through the material and recorded the received signal.
  • Compared waveform attenuation and signal changes to identify internal damage.
  • Applied experimental data collection and analysis techniques relevant to aerospace airframe inspection and fatigue monitoring.
Piezoelectric SensingMATLABSignal AnalysisNon-Destructive Testing

Engineering value

Damage detection without disassembly

Active structural health monitoring can reveal changes in a load path or material condition while leaving the structure intact, supporting faster inspection and condition-based maintenance strategies.
